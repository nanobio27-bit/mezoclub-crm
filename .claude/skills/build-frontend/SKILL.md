# Фронтенд-страница: $ARGUMENTS

Создай React-страницу в `src/client/pages/$ARGUMENTS.tsx`

## Дизайн-система MezoClub
- Фон: `#0B0E14` (bg-[#0B0E14])
- Карточки: `bg-white/5 backdrop-blur-sm border border-white/10 rounded-xl`
- Акцент 1: `#8B5CF6` (фиолетовый — для AI-элементов)
- Акцент 2: `#06B6D4` (голубой — для ссылок/действий)
- Успех: `#10B981` (зелёный — для прибыли, подтверждений)
- Опасность: `#EF4444` (красный — для ошибок, удаления)
- Текст: `text-white` основной, `text-gray-400` вторичный
- Числа и метрики: `font-mono` (JetBrains Mono)
- Обычный текст: `font-sans` (Inter)
- На мобильных: НЕ используй backdrop-blur, замени на bg-[#0B0E14]/95

## Что включить
1. **Заголовок страницы** с количеством записей
2. **Поиск** — текстовое поле с debounce 300ms
3. **Таблица/список** — с сортировкой, пагинацией
4. **Кнопка "Создать"** — открывает модальное окно
5. **Модальное окно** — для создания/редактирования
6. **Подтверждение удаления** — диалог "Вы уверены?"
7. **Loading состояние** — скелетон при загрузке
8. **Пустое состояние** — красивая заглушка когда нет данных
9. **Обработка ошибок** — toast уведомления

## Локализация — ОБЯЗАТЕЛЬНО
- Импортируй `useTranslation` из `react-i18next`
- ВСЕ тексты через `t('ключ')`, НИКОГДА хардкод строк
- Словари: `src/client/locales/ru.json` и `uk.json` — уже готовы
- Если нужен новый ключ — добавь в ОБА файла
- Числа и валюту форматируй: `1 234,56 ₴`

```tsx
// В каждом компоненте:
const { t } = useTranslation();
return <h1>{t('clients.title')}</h1>  // «Клиенты» или «Клієнти»
```

## Технологии
- React hooks: useState, useEffect
- TanStack Query для API-запросов (useQuery, useMutation)
- Tailwind CSS для стилей
- lucide-react для иконок
- axios instance из `../api/client`

## Правила
- Комментарии на русском
- Компонент по умолчанию (default export)
- Responsive: на мобильных таблица → карточки
- Все числа в JetBrains Mono
